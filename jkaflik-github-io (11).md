# Visualisation with Foxglove Studio [​](https://jkaflik.github.io/OpenMowerNext/visualisation.html\#visualisation-with-foxglove-studio)

## Overview [​](https://jkaflik.github.io/OpenMowerNext/visualisation.html\#overview)

![Foxglove with OpenMowerNext visualisation](https://jkaflik.github.io/OpenMowerNext/assets/foxglove.a5it9kx7.png)

[Foxglove Studio](https://foxglove.dev/) is a visualisation tool for robotics developers. It's designed to be easy to use, and to work with any ROS system. Alternatively RViz2 can be used for visualisation.

## Installation [​](https://jkaflik.github.io/OpenMowerNext/visualisation.html\#installation)

Download the latest release from the [Foxglove Studio](https://foxglove.dev/download) website.

## Usage [​](https://jkaflik.github.io/OpenMowerNext/visualisation.html\#usage)

Connect to your ROS system by using Foxglove Websocket connection. This project comes with [Foxglove bridge](https://foxglove.dev/docs/studio/connection/using-foxglove-bridge) installed. Instructions are available [here](https://foxglove.dev/docs/studio/connection/ros2#foxglove-websocket).
